# 侧边信息栏 Chrome 插件

一个简单的 Chrome 浏览器插件，可以在任何网页右侧显示一个信息面板。

## 功能特点

- 🎨 美观的渐变色设计
- 📱 固定在页面右侧
- ⏰ 实时显示当前时间
- 📄 显示当前页面信息（标题、URL）
- 🔽 可收起/展开
- 📱 响应式设计

## 安装方法

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本插件所在的文件夹

## 使用说明

- 插件安装后会自动在所有网页右侧显示信息面板
- 点击右上角的 × 按钮可以关闭侧边栏
- 关闭后，点击右侧悬浮的 📋 按钮可以重新打开

## 自定义

你可以修改以下文件来自定义插件：

- `content.js` - 修改显示的内容和功能
- `styles.css` - 修改样式和外观
- `manifest.json` - 修改插件配置

## 文件结构

```
.
├── manifest.json    # 插件配置文件
├── content.js       # 内容脚本
├── styles.css       # 样式文件
└── README.md        # 说明文档
```
